\begin{abstract}
研究了反对称破缺和超强耦合条件下量子光学的Jaynes-Cummings模型。利用广义多光子旋转波近似，研究了薛定谔方程的平稳解。结果表明，该问题可简化为两个位置-位移谐振子的谐振相互作用。给出了广义哈密顿量的特征态和特征值的显式表达式。我们用解析和数值考虑了两能级系统初始群体和直接多光子共振耦合下的光子分布振荡、坍塌和函数来举例说明我们的物理模型。

\end{abstract}

\section{Introduction}

与量子谐振子耦合的两能级系统（如单辐射模式）作为一个简单而易于处理的模型，在从量子光学到凝聚态物理的许多当代物理分支中发挥了核心作用。在量子光学中，它描述了一个两能级原子与单模电磁辐射[1]共振耦合，即所谓的JC模型[2]。它准确地描述了量子信息学[3]的捕获离子实验。在凝聚态物理我们可能包括Holstein模型[4]，石墨烯在磁场[5]或量子化单模辐射场[6]，量子点耦合光子腔[7]，和电路量子电动力学（QED）设置超导量子位耦合到微波腔[8]。尽管上述系统的基本设置不同，但物理原理类似于腔QED，在那里已经做了第一个实验来实现JC模型[9]。腔体QED可分为弱、强和超强三种耦合机制。对于弱耦合，原子-光子的相互作用率小于原子场和腔场的衰减率。在这种情况下，我们可以通过调整离散腔模式[10]来控制自发发射速率和真空水平。在强耦合状态下，当发射-光子相互作用大于组合衰减速率时，发生拉比振荡形式的相干周期能量交换，而不是不可逆自发发射过程[11]。由于最近在固态半导体[12]或超导体系统[13]方面的成就，人们可以实现超强耦合状态，其中耦合强度与模式频率的可观比例相当。在这种情况下，新的非线性现象是可见的[14]。此外，在这些设置中，我们可以丰富传统的JC模型，包括在传统的空腔QED设置中无法达到的新的交互项。反转对称破缺是JC模型中一个新的破缺（ISB）。因此，在传统的JC模型以及具有经典辐射场的拉比模型中，我们假设偶极矩算子的对角矩阵元素为零，即状态具有一定的空间奇偶性，且能级不退化。然而，在各种感兴趣的系统中，ISB都有内在或外在的原因。系统的反转对称性可以被系统的哈密顿量打破，或者稳态可能不具有这种对称性。如参考文献中所示。[15,16]，当量子系统在稳态下具有永久偶极矩，或者能级随轨道动量退化时，在强激光场作用下系统的量子动力学中出现了新的多光子效应。此外，这些系统具有优势，它允许我们产生拉比频率[17]的辐射，并通过光脉冲[18]产生中等高次谐波。对于半导体版本的JC模型，我们可以通过不对称量子点[19]来实现ISB。在电路QED设置中，它自然地是内部不对称的结果。对于磁通量子比特，势域被简化为双阱势[13]，对于库珀对盒，ISB发生在远离电荷简并点[20]的位置。因此，研究ISB在拉比模型量化版本上的结果是很有意义的，其中拉比模型在超强耦合状态下可能存在多光子效应。

本文研究了ISB对两能级系统与量子化谐振子相互作用的量子动力学的影响。特别地，我们考虑了ISB对广义JC哈密顿量的本征态和本征能量的影响，以及对拉比振荡、坍缩和复兴的动力学的影响。结果表明，与传统的JC系统相比，ISB极大地改变了系统的动力学。类似于准经典情况[16]，它可能是直接的多光子跃迁，因此，在发射体和辐射（玻色子）场之间存在几个光子周期性交换的拉比振荡。我们考虑超强耦合状态。因此，利用共振近似研究了所考虑的系统的量子动力学。

本文的组织结构如下。第二部分，在共振近似的范围内，给出了模型哈密顿量并对角化。第三部分，我们考虑了所考虑的系统的时间量子动力学，并给出了相应的数值模拟。最后在第四部分给出结论。

\section{基本模型哈密顿量和缀饰态图片}

假设一个ISB与量子谐振子耦合的两能级系统，模型哈密顿量可以写成

\begin{equation}
\widehat{H} = \hbar \omega (\widehat{a}^{\dagger} \widehat{a}+\frac{1}{2})+\frac{\hbar \omega _{0}}{2} \widehat\sigma _{z}

+\hbar (-\lambda _{g} \widehat{\sigma _{\downarrow}}+\lambda _{e} \widehat{\sigma _{\uparrow}}+\lambda _{eg} \widehat{\sigma _{x}})(\widehat{a}^{\dagger }+\widehat{a})

\end{equation}

在等式中的前两项(1)分别对应于频率为ω的自由谐振子和一个具有跃迁频率为ω0的二能级系统。最后一项给出了振荡器和一个两能级系统之间的相互作用。创造和湮灭算符，$\widehat{a}^{\dagger }$和$\widehat{a}$，满足玻色子对易规则，$\widehat{\sigma}{x}$,$\widehat{\sigma}{z}$是泡利算符，$\widehat{\sigma}{\uparrow}=(\widehat{I} + \widehat{\sigma }{z})/2$,$\widehat{\sigma}{\downarrow}=(\widehat{I} - \widehat{\sigma }{z})/2$是投影算子，是IBS的结果。这些项将所考虑的系统与传统的JC模型区分开来。当$\lambda _{g}$和$\lambda _{e}$都等于0的时候，我们将得到具有耦合$\hbar$的JC模型（也包括反向旋转项）的通常哈密顿量。在原子或分子和量子点的情况下，$\lambda _{g}$和$\lambda _{e}$对应不定奇偶态的平均偶极矩，而$\lambda _{eg}$对应跃迁偶极矩。在电路量子电动力学的情况下，可在参考论文[13,20]中查阅。不失一般性，我们假设基态和激发态具有相反符号的平均偶极矩，以及$\lambda _{e}$和$\lambda _{g}$都大于0。

首先，我们先将哈密顿量对角化，在缀饰态图片中非常简明。作为一个JC模型，我们的模型不允许精确的解析解。解决JC模型的最强大的近似之一是共振或所谓的旋转波近似（RWA），在近共振$\left | \omega _{0} - \omega \right | \ll \omega $和两个系统之间有弱耦合$\lambda _{eg} \ll \omega _{0}$有效[1]。对于我们的模型，需要一个广义的多光子旋转波近似(RWA)。第一步是改写哈密顿量的形式：

\begin{equation}
\widehat{H} = \widehat{H}_{0}+\widehat{V}
\end{equation}

里面有：

\begin{equation}
\widehat{H}{0}=\widehat{H}{\uparrow} \otimes \widehat{\sigma }{\uparrow}+ \widehat{H}{\downarrow} \otimes \widehat{\sigma }_{\downarrow}
\end{equation}

表示两个非耦合的位置位移振荡器

\begin{equation}
\widehat{H}_{\uparrow}=\hbar \omega (\widehat{a}^{\dagger} \widehat{a}+\frac{1}{2})+\frac{\hbar \omega _{0}}{2}+\hbar \lambda _{e}(\widehat{a}^{\dagger}+\widehat{a})
\end{equation}

\begin{equation}
\widehat{H}_{\downarrow}=\hbar \omega (\widehat{a}^{\dagger} \widehat{a}-\frac{1}{2})+\frac{\hbar \omega _{0}}{2}-\hbar \lambda _{g}(\widehat{a}^{\dagger}+\widehat{a})
\end{equation}

另外有：

\begin{equation}
\widehat{V} = \hbar \lambda {e}\widehat{\sigma }{x}(\widehat{a}^{\dagger}+\widehat{a})
\end{equation}

是相互作用部分。哈密顿量（4）和（5）可以精确对角化。图1给出了具有耦合$\widehat{V}$的两个位置位移谐振子的示意图。很容易看出，在每一个势阱中的本征态是

\begin{equation}
| \uparrow,N^{\lambda _{e}} \rangle \equiv | \uparrow \rangle \otimes e ^{-\frac{\lambda _{e}}{ \omega } (\widehat{a}^{\dagger}-\widehat{a})} |N\rangle
\end{equation}

\begin{equation}
| \downarrow,N^{\lambda _{g}} \rangle \equiv | \downarrow \rangle \otimes e ^{\frac{\lambda _{g}}{ \omega } (\widehat{a}^{\dagger}-\widehat{a})} |N\rangle
\end{equation}

能量是：

\begin{equation}
E^{(0)}_{eN}=\frac{\hbar \omega {0}}{2}+\hbar \omega (N+\frac{1}{2})-\hbar \frac{\lambda ^{2}{e}}{\omega }
\end{equation}

\begin{equation}
E^{(0)}_{gN}=-\frac{\hbar \omega {0}}{2}+\hbar \omega (N+\frac{1}{2})-\hbar \frac{\lambda ^{2}{g}}{\omega }
\end{equation}

$D(\alpha )=e^{\alpha (\widehat{a}^{\dagger} - \widehat{a})}$是位移算符，N=0,1….是量子数。$| \uparrow \rangle$和$| \downarrow \rangle$是$\widehat{\sigma }_{z}$的本征态，$| N^{(\lambda _{e})} \rangle$和$| N^{(\lambda _{g})} \rangle$是位置位移福克态：

\begin{equation}
| N^{(\lambda {e})} \rangle = e^{-(\frac{\lambda {e}}{\omega })(\widehat{a}^{\dagger} -\widehat{a}) }|N \rangle=\sum{M}I{N,M}(\frac{\lambda ^{2}_{e}}{\omega ^{2}})|M \rangle
\end{equation}

\begin{equation}
| N^{(\lambda {g})} \rangle = e^{(\frac{\lambda {g}}{\omega })(\widehat{a}^{\dagger} -\widehat{a}) }|N \rangle=\sum{M}I{M,N}(\frac{\lambda ^{2}_{g}}{\omega ^{2}})|M \rangle
\end{equation}

$I_{N,M}(\alpha )$是拉盖尔函数，通过广义拉盖尔多项式$L^{l}_{n}(\alpha)$定义：

\begin{equation}
I_{s,s^{‘}}(\alpha )=\sqrt{\frac{s^{‘}!}{s!}}e^{-\alpha /2}\alpha ^{(s-s^{‘})/2}L^{s-s^{‘}}_{s^{‘}}(\alpha )

=(-1)^{s-s^{'}}I_{s^{'},s}(\alpha  )


L^{l}_{n}(\alpha )=\frac{1}{n!}e^(\alpha )\frac{d^{n}}{d\alpha ^{n}}(e^{-\alpha }\alpha ^{n+l})

\end{equation}

特别的， $|0^{(\lambda {e})} \rangle$和$ |0^{(\lambda {g})} \rangle$是平均光子数为$\frac{\lambda^{2}{e}}{\omega}$和$\frac{\lambda ^{2}{g}}{\omega } $的格劳伯或相干态。因此我们有两个可以被能量转换的途径：

\begin{equation}
\hbar \omega {eg}=\hbar(\omega {0}+\lambda ^{2}{g}/\omega -\lambda ^{2}{e}/\omega )
\end{equation}
